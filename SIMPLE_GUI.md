# Simple Clean GUI

## Features
- ✅ **Simple & Clean** - Minimal design
- ✅ **Small** - 300x400 pixels
- ✅ **Top Center** - Stays at top of screen
- ✅ **Sticky** - Always on top
- ✅ **Solid Colors** - Green (#00ff00) on dark (#1a1a1a)
- ✅ **Draggable** - Move anywhere

## Layout
```
┌─────────────────────┐
│ 🤖 AI           × │
├─────────────────────┤
│ [Enter command...]  │
│ [Execute]           │
├─────────────────────┤
│ Activity            │
│ [10:30] ▶ hi       │
│ [10:31] └─ Hello   │
└─────────────────────┘
```

## Launch
```bash
# Close old GUI first if running
python3 main.py --gui
```

## Colors
- Background: Dark gray `#1a1a1a`
- Primary: Bright green `#00ff00`  
- Text: White `#ffffff`
- Dim: Gray `#666666`
