# AI Laptop Control Agent

Vision-language agent that autonomously controls a laptop by interpreting screen content and executing user commands through natural language.

## Features

- Screen capture and visual understanding via vision-language models
- Natural language command interpretation
- Automated mouse and keyboard control
- Task chaining for multi-step operations
- Safety guardrails to prevent destructive actions
- Action logging and replay

## Tech Stack

Python, OpenCV, LLM APIs, PyAutoGUI, Pillow

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Laptop-Control-Agent.git
cd AI-Laptop-Control-Agent
pip install -r requirements.txt
```

### Usage

```bash
python agent.py
```
